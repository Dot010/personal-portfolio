# 🚀 Personal Portfolio - Jonathan Carvalho

Este é o meu portfólio profissional, desenvolvido para exibir meus projetos, habilidades e trajetória como desenvolvedor Full-Stack. O foco principal foi criar uma interface moderna, responsiva e com alta performance utilizando as melhores práticas do ecossistema React.

## 🛠️ Tecnologias Utilizadas

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
* **Animações:** [Framer Motion](https://www.framer.com/motion/)
* **Componentes de UI:** [Shadcn UI](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
* **Slider/Carrossel:** [Swiper.js](https://swiperjs.com/)
* **Ícones:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)

---

## ✨ Funcionalidades

* **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.
* **Modo Dark Nativo:** Estética profissional focada em legibilidade e modernidade.
* **Seção de Projetos Dinâmica:** Slider interativo com Swiper.js para navegação suave entre os cases de sucesso.
* **Animações de Entrada:** Transições fluidas de página e componentes utilizando Framer Motion.
* **Acessibilidade:** Uso de componentes semânticos e otimização para leitores de tela.

---

## 🚀 Desafios Técnicos Superados

Durante o desenvolvimento, foquei em resolver problemas reais de arquitetura Frontend:
1.  **Configuração do Swiper no Next.js:** Implementação de `transpilePackages` e declarações globais de tipos para garantir a compatibilidade do carrossel no build de produção.
2.  **Performance de Imagens:** Utilização do componente `next/image` para otimização automática, prioridade de carregamento (LCP) e ajuste de escala (`object-cover`).
3.  **Gerenciamento de Estado de UI:** Criação de componentes reutilizáveis e tipagem rigorosa com TypeScript para evitar erros em tempo de execução.

## 📦 Como rodar o projeto localmente

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Dot010/personal_landing_page.git](https://github.com/Dot010/personal_landing_page.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
4.  **Acesse no navegador:** `http://localhost:3000`
